Why

One row per window

Five Arc windows? Five rows. Jump to the right one in a keystroke.

Type to filter

Fuzzy match across app names and window titles.

Native & tiny

Swift, ~200 KB binary, arm64. Zero overhead when idle.

Open source

MIT. No telemetry, no accounts.

Install

  1. Download the zip, unzip, and drag mwitch.app into /Applications.
  2. Double-click mwitch.app to launch. Signed and notarized by Apple, so no Gatekeeper warnings.
  3. Grant Accessibility permission. A macOS dialog will pop up. Click Open System Settings, then toggle mwitch on in the list. Lets mwitch intercept Cmd + Tab and raise the window you pick.
  4. Grant Screen Recording permission. A second dialog will pop up with the same flow: Open System Settings, toggle mwitch on. Required by macOS for any app that reads window titles. mwitch never records your screen.
  5. When macOS asks to Quit & Reopen mwitch, click that. Permissions only activate after a relaunch.
  6. Press Cmd + Tab. The switcher should pop up. Hold Cmd, tap Tab to advance, release to switch. mwitch starts automatically at login from now on.

Stuck? If a permission dialog disappeared before you could click it, open the panes manually from the mwitch menu bar icon → Open Accessibility Settings… and Open Screen Recording Settings…. If mwitch already appears in either list with the toggle on but Cmd+Tab still doesn't work, remove it with the – button, then click + and re-add /Applications/mwitch.app.
